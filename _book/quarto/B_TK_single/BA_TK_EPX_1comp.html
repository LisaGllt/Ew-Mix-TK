4.1.1 Data importation

Notations :

  • Date_start : Start of the experiement (t0)
  • Molecule : Substance to which the earthworm is exposed
  • Dose : Exposure dose (mg/kg)
  • ID_recipient : Recipient in which the soil was prepared
  • ID_camp : ID of the earthworm campagne and generation (see XXX for more information)
  • ID : ID of the earthworm
  • Nb_rep : Identification number of the replicate
  • Time_point : Time point to which the earthworm is dedicated
  • Phase : Uptake or Elimination phase
  • Time_introduction_D0 : Time of the day when the earthworm was introduced into his contaminated cosm
  • w0 : Body weight at t0
  • w_Tp : Body weight at the given time point
  • w_frozen : Body weight before congelation
  • C_worm : Measured concentration of the substance in the earthworm
  • C_soil : Measured concentration of the substance in the soil at the given time point

df_priors_dist <- MCMC_out_EPX$Priors |> 
  dplyr::select(Nom, Distribution, P1, P2, P3, P4) |> 
  rowwise() |> 
  mutate(samples = case_when(
    Distribution == "Uniform" ~ list(runif(n_samples, min = P1, max = P2)),
    Distribution == "Normal" ~ list(rnorm(n_samples, mean = P1, sd = P2)),
    Distribution == "LogUniform" ~ list(exp(runif(n_samples, min = log(P1), max = log(P2)))),
    Distribution == "TruncNormal" ~ list(rtruncnorm(n_samples, a = P3, b = P4, mean = P1, sd = P2))
