sitess&species.txt ----------------------------
each line is a site, each column 'vtXXXX' represents a species
site: ID of the sampled sites
long2e : longitude in Lambert2etendu Geographic Coordinate System (EPSG:27572)
lat2e : latitude in Lambert2etendu Geographic Coordinate System (EPSG:27572)
Altitude : altitude in meters
land_cover : vegetation cover
vtXXXX : for species names see the file traits.txt

traits.txt ----------------------------
each line is a species, based on Fauna Europaea www.faunaeur.org. This means that we aggregate subspecies into species in some cases.
species_code : code of the species in sites&species.txt
species.name : names of the species according to Fauna Europaea
Other columns describe species caracteristics
